# BreathArt Creative Vision Studio

Welcome to the BreathArt project repository! This project has been fully migrated to a production-ready **Next.js** application.

## Directory Structure

- **`breathart-next/`**: The modern Next.js 16 (App Router) project containing the fully functional and SEO-optimized website. This is the main application folder.
- **`legacy_archive/`**: The original static HTML/CSS/JS files, including legacy assets and scripts. This folder has been added to `.gitignore` to keep the repository clean.

## Getting Started

To run the Next.js application locally:

1. Navigate to the Next.js project folder:
   ```bash
   cd breathart-next
   ```
2. Install the necessary dependencies (if not already installed):
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) in your browser to view the application.

## Technologies Used

- **Next.js 16**: Utilizing the App Router for fast, static server-side rendering and built-in SEO capabilities.
- **React**: Component-based architecture for a clean and maintainable codebase.
- **Custom CSS**: Preserved the original bespoke CSS design system (Vanilla CSS), ensuring 100% fidelity to the original UI/UX.

## Legacy Content

If you ever need to reference the original HTML implementation, all files are safely stored in the `legacy_archive/` directory. Note that changes made there will not reflect in the live Next.js application.
